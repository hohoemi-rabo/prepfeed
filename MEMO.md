# YouTube-Scope 開発メモ

---

## Phase 1: 基盤機能（完了）

### 概要
YouTubeチャンネル分析とキーワード検索による動画企画支援ツール

### 技術スタック
| 技術 | バージョン | 用途 |
|------|-----------|------|
| Next.js | 15.5.7 | App Router, API Routes |
| React | 19.1.0 | UI |
| TypeScript | 5 | 型安全性 |
| Tailwind CSS | 3.4.17 | スタイリング |
| Zustand | 5.0.8 | 状態管理（ソート設定） |
| Recharts | 3.2.1 | グラフ描画 |
| Vercel KV | 3.0.0 | キャッシュ（本番環境） |

### 機能一覧

#### 1. チャンネル分析（赤テーマ）
- **ページ**: `/` → `/channel/[id]`
- **検索**: チャンネル名オートコンプリート
- **表示内容**:
  - チャンネル情報（登録者数、総視聴回数）
  - 最新50本の動画一覧
  - 視聴回数推移グラフ
- **機能**:
  - 5種類のソート
  - SNSシェア（動的OGP画像）

#### 2. キーワード検索（青テーマ）
- **ページ**: `/` → `/keyword/[query]`
- **検索**: キーワード入力
- **表示内容**:
  - 視聴回数順に上位50本
  - タグ表示（最大8個、クリックで再検索）
- **機能**:
  - 5種類のソート

#### 3. ソート機能
| タイプ | 説明 |
|--------|------|
| views | 視聴回数順 |
| date | 投稿日順 |
| growth | 成長率順（1日あたりの視聴回数） |
| comments | コメント数順 |
| likes | いいね数順 |

#### 4. 動画カード
- サムネイル、タイトル、統計情報
- バッジ: 「新着」（7日以内）、「急上昇」
- タグ表示（最大8個）
- 日本語数値フォーマット（万/億）

### API構成
| エンドポイント | 用途 | Quota | キャッシュ |
|---------------|------|-------|-----------|
| `/api/youtube/search` | チャンネル検索 | 100 | 30分 |
| `/api/youtube/channel/[id]` | チャンネル詳細 | 103 | 30分 |
| `/api/youtube/keyword` | キーワード検索 | 150 | 30分 |
| `/api/og` | OGP画像生成 | - | - |

### その他ページ
- `/contact` - お問い合わせ
- `/disclaimer` - 免責事項
- `/privacy` - プライバシーポリシー

### 制限・保護
- レート制限: 10リクエスト/分/IP
- キャッシュ: 30分TTL（Vercel KV）
- エラーハンドリング: 6種類、指数バックオフ
