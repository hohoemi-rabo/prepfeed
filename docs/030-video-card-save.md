# 030: 動画カードへの保存機能追加

## 概要
動画カードコンポーネント（`VideoCard.tsx`）に動画保存ボタンを追加する。

## 要件
- カード右上に保存アイコンを配置
- 保存/解除が即時反映
- チャンネル詳細ページ・キーワード結果ページ両方で動作
