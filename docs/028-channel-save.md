# 028: チャンネル詳細ページへの保存機能追加

## 概要
チャンネル詳細ページ（`/channel/[id]`）にチャンネル保存ボタンを追加する。

## 要件
- チャンネル名付近に保存ボタンを配置
- 保存/解除が即時反映
- ページリロード後も保存状態を維持
