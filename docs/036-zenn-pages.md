# 036: Zennページ（ユーザー記事・キーワード検索）

## 概要

Zenn のユーザー記事一覧ページとキーワード（トピック）検索結果ページを実装する。

## 背景

Zenn APIクライアント（033）と記事コンポーネント（034）を組み合わせてページを構築する。

## 要件

### ユーザー記事ページ（`/zenn/user/[username]`）

- [ ] ユーザー情報カード（アバター、名前、記事数）
- [ ] 記事一覧（ArticleList 使用）
- [ ] ソートタブ（いいね数 / 投稿日）
- [ ] ローディング状態表示
- [ ] エラー状態表示（ユーザー未検出等）
- [ ] 「Zennで見る」外部リンク

### キーワード検索結果ページ（`/zenn/keyword/[query]`）

- [ ] 検索トピック名表示 + 件数
- [ ] キーワード再検索入力フォーム
- [ ] 記事一覧（ArticleList 使用）
- [ ] ソートタブ（いいね数 / 投稿日）
- [ ] ローディング状態表示
- [ ] 検索結果0件時のメッセージ

### ホームページからの導線

- [ ] Zennユーザー検索フォーム → `/zenn/user/[username]`
- [ ] Zennキーワード検索フォーム → `/zenn/keyword/[query]`

### メタデータ

- [ ] 各ページのタイトル・OGP設定

## 受け入れ条件

- [ ] `/zenn/user/catnose99` でユーザー記事一覧が表示される
- [ ] `/zenn/keyword/Next.js` でトピック検索結果が表示される
- [ ] ソート切替が正しく動作する
- [ ] ホームページからZenn検索が可能
- [ ] エラー状態が適切に表示される
- [ ] レスポンシブ対応

## 依存関係

- 033（Zenn APIクライアント & APIルート）
- 034（記事共通コンポーネント）
