# 036: Zennページ（ユーザー記事・キーワード検索）

## 概要

Zenn のユーザー記事一覧ページとキーワード（トピック）検索結果ページを実装する。

## 背景

Zenn APIクライアント（033）と記事コンポーネント（034）を組み合わせてページを構築する。

## 要件

### ユーザー記事ページ（`/zenn/user/[username]`）

- [x] ユーザー情報カード（アバター、名前、記事数）
- [x] 記事一覧（ArticleList 使用）
- [x] ソートタブ（いいね数 / 投稿日）
- [x] ローディング状態表示
- [x] エラー状態表示（ユーザー未検出等）
- [x] 「Zennで見る」外部リンク

### キーワード検索結果ページ（`/zenn/keyword/[query]`）

- [x] 検索トピック名表示 + 件数
- [x] キーワード再検索入力フォーム
- [x] 記事一覧（ArticleList 使用）
- [x] ソートタブ（いいね数 / 投稿日）
- [x] ローディング状態表示
- [x] 検索結果0件時のメッセージ

### ホームページからの導線

- [x] Zennユーザー検索フォーム → `/zenn/user/[username]`
- [x] Zennキーワード検索フォーム → `/zenn/keyword/[query]`

### メタデータ

- [x] 各ページのタイトル・OGP設定
