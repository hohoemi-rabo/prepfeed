# 022: キーワード検索ページUI実装

## 概要
キーワード検索結果を表示する専用ページ `/keyword/[query]` を実装する。既存のVideoCard、SortTabs、VideoListコンポーネントを再利用し、統一感のあるUIを提供する。

## 要件
- URL: `/keyword/[query]`
- 既存コンポーネントの最大限の再利用
- レスポンシブデザイン対応
- ソート機能の統合
- ローディング・エラー状態の適切な表示
