# 024: VideoCardにタグ表示追加

## 概要
VideoCardコンポーネントにYouTube動画のタグ（tags）を表示する機能を追加。キーワード検索時に、人気動画がどのようなタグを使っているか確認できるようにし、動画企画のヒントを提供する。

## 要件
- タグ情報の表示（最大5-10個程度）
- 既存のVideoCardデザインとの調和
- タグのクリックで検索できる（オプション）
- レスポンシブ対応
