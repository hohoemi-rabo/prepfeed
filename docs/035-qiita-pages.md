# 035: Qiitaページ（ユーザー記事・キーワード検索）

## 概要

Qiita のユーザー記事一覧ページとキーワード検索結果ページを実装する。

## 背景

Qiita APIクライアント（032）と記事コンポーネント（034）を組み合わせてページを構築する。

## 要件

### ユーザー記事ページ（`/qiita/user/[id]`）

- [ ] ユーザー情報カード（アイコン、名前、記事数、フォロワー数）
- [ ] 記事一覧（ArticleList 使用）
- [ ] ソートタブ（いいね数 / 投稿日 / ストック数）
- [ ] ローディング状態表示
- [ ] エラー状態表示（ユーザー未検出等）
- [ ] 「Qiitaで見る」外部リンク

### キーワード検索結果ページ（`/qiita/keyword/[query]`）

- [ ] 検索キーワード表示 + 件数
- [ ] キーワード再検索入力フォーム
- [ ] 記事一覧（ArticleList 使用）
- [ ] ソートタブ（いいね数 / 投稿日 / ストック数）
- [ ] ローディング状態表示
- [ ] 検索結果0件時のメッセージ

### ホームページからの導線

- [ ] Qiitaユーザー検索フォーム → `/qiita/user/[id]`
- [ ] Qiitaキーワード検索フォーム → `/qiita/keyword/[query]`

### メタデータ

- [ ] 各ページのタイトル・OGP設定

## 受け入れ条件

- [ ] `/qiita/user/mizchi` で記事一覧が表示される
- [ ] `/qiita/keyword/React` で検索結果が表示される
- [ ] ソート切替が正しく動作する
- [ ] ホームページからQiita検索が可能
- [ ] エラー状態が適切に表示される
- [ ] レスポンシブ対応

## 依存関係

- 032（Qiita APIクライアント & APIルート）
- 034（記事共通コンポーネント）
