# 035: Qiitaページ（ユーザー記事・キーワード検索）

## 概要

Qiita のユーザー記事一覧ページとキーワード検索結果ページを実装する。

## 背景

Qiita APIクライアント（032）と記事コンポーネント（034）を組み合わせてページを構築する。

## 要件

### ユーザー記事ページ（`/qiita/user/[id]`）

- [x] ユーザー情報カード（アイコン、名前、記事数、フォロワー数）
- [x] 記事一覧（ArticleList 使用）
- [x] ソートタブ（いいね数 / 投稿日 / ストック数）
- [x] ローディング状態表示
- [x] エラー状態表示（ユーザー未検出等）
- [x] 「Qiitaで見る」外部リンク

### キーワード検索結果ページ（`/qiita/keyword/[query]`）

- [x] 検索キーワード表示 + 件数
- [x] キーワード再検索入力フォーム
- [x] 記事一覧（ArticleList 使用）
- [x] ソートタブ（いいね数 / 投稿日 / ストック数）
- [x] ローディング状態表示
- [x] 検索結果0件時のメッセージ

### ホームページからの導線

- [x] Qiitaユーザー検索フォーム → `/qiita/user/[id]`
- [x] Qiitaキーワード検索フォーム → `/qiita/keyword/[query]`

### メタデータ

- [x] 各ページのタイトル・OGP設定

## 受け入れ条件

- [x] `/qiita/user/mizchi` で記事一覧が表示される
- [x] `/qiita/keyword/React` で検索結果が表示される
- [x] ソート切替が正しく動作する
- [x] ホームページからQiita検索が可能
- [x] エラー状態が適切に表示される
- [x] レスポンシブ対応

## 依存関係

- 032（Qiita APIクライアント & APIルート）
- 034（記事共通コンポーネント）
