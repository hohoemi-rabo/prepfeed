# 039: 監視設定UI（ステップウィザード）

## 概要

監視設定の追加・編集・削除を行うUIを実装する。ステップ形式のウィザードで設定を追加する。

## 背景

有料ユーザーが直感的に監視対象を設定できるUIが必要。ステップウィザードで段階的に入力を進める。

## 要件

### 監視設定追加ウィザード

#### Step 1: プラットフォーム選択

- [ ] YouTube / Qiita / Zenn の3択カード
- [ ] 各プラットフォームのアイコン + 説明
- [ ] 選択時にハイライト表示

#### Step 2: タイプ選択

- [ ] プラットフォームに応じた選択肢
  - YouTube: キーワード / チャンネル
  - Qiita: キーワード / ユーザー
  - Zenn: キーワード / ユーザー
- [ ] 各タイプの説明文表示

#### Step 3: 検索・入力

- [ ] キーワードタイプ: テキスト入力 → そのまま登録
- [ ] チャンネル/ユーザータイプ: 検索入力 → 候補から選択
  - YouTube: 既存の SearchBar パターン流用
  - Qiita: ユーザーID入力
  - Zenn: ユーザー名入力

#### Step 4: 取得件数選択

- [ ] Lite (50件) / Deep (100件) / Max (200件) の3択
- [ ] 各プランの説明表示

#### Step 5: 確認 & 登録

- [ ] 入力内容の確認画面
- [ ] 「登録」ボタンで POST /api/settings 実行
- [ ] 初回取得の進捗表示（ローディング）
- [ ] 完了通知

### 監視設定一覧（ダッシュボード内）

- [ ] カード形式で設定を一覧表示
- [ ] 各カードに表示:
  - プラットフォームアイコン
  - タイプ（キーワード/チャンネル/ユーザー）
  - 値（検索ワード or チャンネル名）
  - 取得件数
  - 有効/無効状態
  - 最終取得日時
- [ ] 編集ボタン（取得件数・有効/無効の変更）
- [ ] 削除ボタン（確認ダイアログ付き）

### 編集モーダル

- [ ] 取得件数の変更（50/100/200）
- [ ] 有効/無効の切替
- [ ] 表示名の変更

## 受け入れ条件

- [ ] ステップウィザードで監視設定を追加できる
- [ ] 各ステップで「戻る」が機能する
- [ ] 初回取得の完了が表示される
- [ ] 設定一覧にカードが追加される
- [ ] 編集モーダルで設定を変更できる
- [ ] 削除確認後に設定が削除される
- [ ] レスポンシブ対応
