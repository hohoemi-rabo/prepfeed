# 039: 監視設定UI（ステップウィザード）

## 概要

監視設定の追加・編集・削除を行うUIを実装する。ステップ形式のウィザードで設定を追加する。

## 背景

有料ユーザーが直感的に監視対象を設定できるUIが必要。ステップウィザードで段階的に入力を進める。

## 要件

### 監視設定追加ウィザード

#### Step 1: プラットフォーム選択

- [x] YouTube / Qiita / Zenn の3択カード
- [x] 各プラットフォームのアイコン + 説明
- [x] 選択時にハイライト表示

#### Step 2: タイプ選択

- [x] プラットフォームに応じた選択肢
  - YouTube: キーワード / チャンネル
  - Qiita: キーワード / ユーザー
  - Zenn: キーワード / ユーザー
- [x] 各タイプの説明文表示

#### Step 3: 検索・入力

- [x] キーワードタイプ: テキスト入力 → そのまま登録
- [x] チャンネル/ユーザータイプ: 検索入力 → 候補から選択
  - YouTube: 既存の SearchBar パターン流用
  - Qiita: ユーザーID入力
  - Zenn: ユーザー名入力

#### Step 4: 取得件数選択

- [x] Lite (50件) / Deep (100件) / Max (200件) の3択
- [x] 各プランの説明表示

#### Step 5: 確認 & 登録

- [x] 入力内容の確認画面
- [x] 「登録」ボタンで POST /api/settings 実行
- [x] 初回取得の進捗表示（ローディング）
- [x] 完了通知

### 監視設定一覧（ダッシュボード内）

- [x] カード形式で設定を一覧表示
- [x] 各カードに表示:
  - プラットフォームアイコン
  - タイプ（キーワード/チャンネル/ユーザー）
  - 値（検索ワード or チャンネル名）
  - 取得件数
  - 有効/無効状態
  - 最終取得日時
- [x] 編集ボタン（取得件数・有効/無効の変更）
- [x] 削除ボタン（確認ダイアログ付き）

### 編集モーダル

- [x] 取得件数の変更（50/100/200）
- [x] 有効/無効の切替
- [x] 表示名の変更

## 受け入れ条件

- [x] ステップウィザードで監視設定を追加できる
- [x] 各ステップで「戻る」が機能する
- [x] 初回取得の完了が表示される
- [x] 設定一覧にカードが追加される
- [x] 編集モーダルで設定を変更できる
- [x] 削除確認後に設定が削除される
- [x] レスポンシブ対応
