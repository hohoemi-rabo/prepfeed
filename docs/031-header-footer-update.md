# 031: ヘッダー・フッター・ホームページ更新

## 概要

PrepFeed ブランディングに合わせてヘッダー・フッターを更新し、ホームページを3プラットフォーム（YouTube / Qiita / Zenn）対応にする。

## 背景

Phase 1 のYouTubeスコープから PrepFeed へのリブランドに伴い、レイアウトコンポーネントとホームページの更新が必要。

## 要件

### ヘッダー更新

- [ ] ロゴ・サービス名を「PrepFeed」に変更
- [ ] ナビゲーションリンク更新
  - ホーム
  - ダッシュボード（認証済みユーザーのみ）
  - お問い合わせ
- [ ] SNSリンクは維持（X, Instagram, GitHub）

### フッター更新

- [ ] サービス名・説明文を PrepFeed に変更
- [ ] サブタイトル反映:「集めて、分析して、ネタにする。」
- [ ] リンク更新（免責事項、プライバシー、お問い合わせ）

### ホームページ更新（`/`）

- [ ] 3プラットフォーム対応の検索UI
  - YouTube: チャンネル検索（赤グラデーション）— 既存SearchBar活用
  - YouTube: キーワード検索（赤グラデーション）— 既存input活用
  - Qiita: ユーザー検索 + キーワード検索（緑系グラデーション）
  - Zenn: ユーザー検索 + キーワード検索（青系グラデーション）
- [ ] プラットフォーム選択UI（タブまたはカード形式）
- [ ] 無料機能と有料機能の説明セクション
- [ ] PrepFeed のコンセプト説明

### ブランドカラー追加

- [ ] Qiita: #55C500（緑）系グラデーション
- [ ] Zenn: #3EA8FF（青）系グラデーション
- [ ] CSS変数への追加

## 受け入れ条件

- [ ] ヘッダーに「PrepFeed」ロゴが表示される
- [ ] フッターの説明文が更新されている
- [ ] ホームページで3プラットフォームの検索UIが利用できる
- [ ] YouTube検索は既存機能がそのまま動作する
- [ ] Qiita / Zenn の検索UIが表示される（API接続は別チケット）
- [ ] レスポンシブ対応
