# 029: キーワード結果ページへの保存機能追加

## 概要
キーワード検索結果ページ（`/keyword/[query]`）にキーワード保存ボタンを追加する。

## 要件
- 検索キーワード付近に保存ボタンを配置
- 保存/解除が即時反映
- ページリロード後も保存状態を維持
