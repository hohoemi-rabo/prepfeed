# 031: 保存一覧ページ実装

## 概要
保存したチャンネル・動画・キーワードを一覧表示するページ（`/saved`）を実装する。

## 要件
- タブで「チャンネル / 動画 / キーワード」を切替
- 各保存アイテムをカード形式で表示
- 削除機能（×ボタン）
- 空状態のメッセージ表示

```

```

### カードレイアウト（チャンネル例）
```

│ [サムネイル] チャンネル名      [×]  │
```
